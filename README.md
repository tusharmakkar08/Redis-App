Redis Application
==================

A simple application that pulls records from a mysql database and places it on a queue .

Language Used :
--------------

* Python
* Shell Scripting

Database Used :
---------------

* MySQL
* Redis

Libraries Used :
----------------
	
* redis
* MySQLdb
* sys
* os

How to Run :
-----------
<pre>
				sh automaticscript.sh	
				# Run Script for creating database and table
				python redisapp.py		
				# Running python code
</pre>
